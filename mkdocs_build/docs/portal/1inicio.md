# Portal 4SUP 

## Acesso
<p>
O Portal Web 4SUP é a interface administrativa utilizada pela base operacional para acompanhar e gerenciar as supervisões realizadas em campo. Através do portal, gestores e coordenadores podem visualizar agendas, supervisionar rotas, analisar registros de ocorrência, consultar checklists preenchidos, administrar usuários e acompanhar indicadores de desempenho.<br><br>

A plataforma centraliza todas as informações enviadas pelo aplicativo, permitindo o monitoramento em tempo real das atividades executadas pelos supervisores, bem como o registro histórico de visitas, ações e evidências associadas a cada posto.<br><br>

O Portal Web é acessado através de navegador e utiliza autenticação individual, garantindo segurança e rastreabilidade das operações administrativas.<br><br>
</p>
